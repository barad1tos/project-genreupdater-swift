  <action>
Rewrite `ReportsChangeLog` from Table-based to LazyVStack with pinnedViews for sticky session headers.

**Public API changes:**
Add callback parameters to init (backward-compatible with defaults):
```swift
public init(
    entries: [ChangeLogEntry],
    onUndoEntry: ((ChangeLogEntry) -> Void)? = nil,
    onUndoSession: (([ChangeLogEntry]) -> Void)? = nil
)
```
Store as properties. SharedUI must NOT import Services -- undo is callback-injected (Pattern 3 from RESEARCH.md).

**Session grouping:**
Add a private `SessionGroup` struct:
```swift
private struct SessionGroup: Identifiable {
    let id: Date
    let header: String  // "Feb 23, 2026 -- 14:30 (142 changes)"
    let entries: [ChangeLogEntry]
}
```

Add `groupBySession(_:)` function: sort entries by timestamp descending, group entries within 60-second gaps into the same session. Format header as "MMM d, yyyy -- HH:mm (N changes)" using a static DateFormatter.

**Layout rewrite:**
Replace `Table` with:
```swift
ScrollView {
    LazyVStack(spacing: 0, pinnedViews: [.sectionHeaders]) {
        ForEach(sessionGroups) { group in
            Section {
                ForEach(group.entries) { entry in
                    changeLogRow(entry: entry)
                    Divider()
                }
            } header: {
                sessionHeader(group: group)
            }
        }
    }
}
```

**Session header (sticky):**
- `HStack` with session header text (AppFont.caption, Ayu.fgSecondary) + Spacer + "Undo Session" button (only visible if `onUndoSession` is non-nil)
- Background: `Ayu.bgPrimary` for opaque sticky header look
- Padding: horizontal Spacing.md, vertical Spacing.xs

**Change log row with hover undo:**
- `@State private var hoveredEntryID: UUID?`
- Each row: `HStack` with change type icon + track name/artist VStack + old->new change value + undo button (only when hovered AND `onUndoEntry` is non-nil)
- Undo button: `Image(systemName: "arrow.uturn.backward.circle")` with `Ayu.warning` foreground, `.buttonStyle(.plain)`, `.transition(.opacity)` with `Motion.curveFast`
- `.onHover` sets `hoveredEntryID`
- Row hover highlight: `Ayu.bgTertiary.opacity(0.5)` background when hovered (consistent with SharedUI list row interaction pattern)
- `.contentShape(.rect)` for macOS 15 scroll fix

**Undo confirmation alert:**
- `@State private var showUndoConfirmation = false`
- `@State private var undoConfirmationEntry: ChangeLogEntry?`
- `@State private var undoConfirmationSession: SessionGroup?`
- When undo button tapped, set the entry/session and show confirmation
- Alert: "Revert [changeType] change for [trackName]?" with "Undo" (destructive role) and "Cancel" buttons
- On confirm: call `onUndoEntry?(entry)` or `onUndoSession?(group.entries)`

**Keep existing:**
- Filter bar (search + change type picker) -- adapt to work with grouped view
- `filteredEntries` computed property -- apply filters then group by session
- Empty state for no matching entries (update text to remove "No Changes Found" -- use "No changes match your filters" for filtered, or nothing for the global empty state which is handled by ReportsView)

**Empty state text updates:**

- When filters produce no matches: "No changes match the current filters." (already close, keep this)
