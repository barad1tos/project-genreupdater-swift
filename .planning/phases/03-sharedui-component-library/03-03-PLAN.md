Create `Packages/SharedUI/Sources/SharedUI/Components/HeroGauge.swift` with these components:

**1. ArcShape (private)**
- `private struct ArcShape: Shape` with properties:
  - `progress: Double` (0.0-1.0) -- how far the arc fills
  - `radius: CGFloat` -- the arc radius from center
  - `lineWidth: CGFloat` -- stroke width
- Conform to `Animatable` via `var animatableData: Double { get { progress } set { progress = newValue } }` for smooth interpolation
- `func path(in rect: CGRect) -> Path`:
  - Center point: `CGPoint(x: rect.midX, y: rect.maxY)` -- bottom center of the rect (arcs open upward)
  - Start angle: `.degrees(180)` (left / 9 o'clock)
  - End angle: `.degrees(180 + 180 * progress)` (sweeps clockwise to right / 3 o'clock at 100%)
  - Use `path.addArc(center:radius:startAngle:endAngle:clockwise: false)` -- Core Graphics convention: clockwise=false draws counterclockwise which is visually clockwise on screen (CG Y-axis is flipped)
  - Return the path (stroked externally, not filled)

**2. GaugeLayer enum (private)**
- `private enum GaugeLayer: CaseIterable, Sendable` with cases `.genre`, `.year`, `.consistency`
- Computed properties:
  - `color: Color` -- `.genre: Ayu.accent` (orange), `.year: Ayu.success` (green), `.consistency: Ayu.info` (blue)
  - `label: String` -- "Genre", "Year", "Consistency"

**3. HeroGauge (public)**
- `public struct HeroGauge: View` with parameters:
  - `genreCoverage: Double` (0.0-1.0)
  - `yearCoverage: Double` (0.0-1.0)
  - `consistencyCoverage: Double` (0.0-1.0)
  - `trackCount: Int`
- `public init(genreCoverage:yearCoverage:consistencyCoverage:trackCount:)`
- `@State private var animatedGenre: Double = 0`
- `@State private var animatedYear: Double = 0`
- `@State private var animatedConsistency: Double = 0`
- `@State private var hoveredLayer: GaugeLayer? = nil`

**Arc layout constants (private):**
- Arc line width: 16pt (within the 14-18pt range per decision)
- Gap between concentric arcs: 6pt
- Outer arc radius: computed from available width. Given `let maxRadius = min(geometry.size.width, geometry.size.height * 2) / 2 - lineWidth / 2`, then:
  - Genre (outer): `maxRadius`
  - Year (mid): `maxRadius - lineWidth - gap`
  - Consistency (inner): `maxRadius - 2 * (lineWidth + gap)`

**Body layout:**
- `VStack(spacing: Spacing.md)` containing:
  1. Arc area: `GeometryReader` wrapping a `ZStack` of arcs + center content
  2. Legend: `HStack(spacing: Spacing.lg)` with three legend items

**Arc area (inside GeometryReader):**
- For each layer (genre, year, consistency):
  a. Background track arc: `ArcShape(progress: 1.0, radius: layerRadius, lineWidth: lineWidth).stroke(layer.color.opacity(0.15), style: StrokeStyle(lineWidth: lineWidth, lineCap: .butt))` -- full 180-degree semi-transparent background
  b. Value arc: `ArcShape(progress: animatedValue, radius: layerRadius, lineWidth: lineWidth).stroke(layer.color, style: StrokeStyle(lineWidth: lineWidth, lineCap: .butt))` -- the animated foreground arc

**Center content (overlaid in ZStack):**
- Positioned at center-bottom of the half-circle area using `.position(x: geometry.size.width / 2, y: geometry.size.height)`... or better: use a VStack positioned in the center of the arc opening
- Default (hoveredLayer == nil): show `Text(trackCount.formatted())` in `AppFont.display` + `Text("tracks")` in `AppFont.caption` and `Ayu.fgSecondary`
- When hoveredLayer != nil: show `Text("\(Int(coverageForLayer(hoveredLayer!) * 100))%")` in `AppFont.display` + `Text(hoveredLayer!.label)` in `AppFont.caption` and `Ayu.fgSecondary`
- Animate the switch with `.contentTransition(.numericText())` on the number text and `.animation(Motion.curveFast, value: hoveredLayer)` on the container

**Per-arc hover detection:**
- Use `.onContinuousHover { phase in ... }` (macOS 13+) on the GeometryReader
- In the `.active(let location)` case:
  - Calculate distance from center point `(width/2, height)` to the cursor location
  - Determine which ring the cursor falls in based on distance ranges:
    - Genre (outer): `outerRadius - lineWidth/2 ... outerRadius + lineWidth/2`
    - Year (mid): `midRadius - lineWidth/2 ... midRadius + lineWidth/2`
    - Consistency (inner): `innerRadius - lineWidth/2 ... innerRadius + lineWidth/2`
  - Also verify the cursor is in the upper half (y < center.y) since arcs are half-circles opening upward
  - Set `hoveredLayer` to the matching layer, or nil if not over any arc
- In the `.ended` case: set `hoveredLayer = nil`

**Draw-in animation on appear:**
- `.onAppear { withAnimation(.spring(duration: 0.8, bounce: 0.15)) { animatedGenre = genreCoverage; animatedYear = yearCoverage; animatedConsistency = consistencyCoverage } }`
- Stagger slightly: use `.delay(0.05)` for year, `.delay(0.1)` for consistency for a cascading draw-in effect. Apply delays via separate withAnimation calls or DispatchQueue.main.asyncAfter.

**Legend:**
- `HStack(spacing: Spacing.lg)` with three items, each a `HStack(spacing: Spacing.xxs)`:
  - Colored dot: `Circle().fill(layer.color).frame(width: 8, height: 8)`
  - Label: `Text(layer.label)` in `AppFont.caption`, `Ayu.fgSecondary`
  - Percentage: `Text("\(Int(coverage * 100))%")` in `AppFont.caption.bold()`, `Ayu.fgPrimary`

**Accessibility:**
- `.accessibilityElement(children: .ignore)` on the gauge area
- `.accessibilityLabel("Library health gauge")`
- `.accessibilityValue("Genre \(Int(genreCoverage * 100)) percent, Year \(Int(yearCoverage * 100)) percent, Consistency \(Int(consistencyCoverage * 100)) percent, \(trackCount) tracks")`

**Important notes:**
- NO dependency on Core models -- plain Double/Int only
- Use `Ayu.accent` (orange), `Ayu.success` (green), `Ayu.info` (blue) -- these are the locked decision colors
- Line cap MUST be `.butt` (flat) -- NOT `.round`. The existing ProgressRing uses `.round` but the decision explicitly requires flat caps for technical/minimalist look
- The arc animation uses raw `.spring()` (not `motionAnimation`) because it's an entrance-only animation per research note
