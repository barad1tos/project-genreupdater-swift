**DashboardViewModel.swift:**

1. Add a `shimmerStartTime` property (`private var shimmerStartTime: Date?`) to track when shimmer began. Set it in `loadCachedMetrics` when entering `.shimmer` state (`shimmerStartTime = Date()`).

2. Add computed properties for view state:
   - `var showShimmer: Bool` -- returns `true` when `loadingState == .shimmer`
   - `var showLiveContent: Bool` -- returns `true` when loadingState is `.cached`, `.updating`, or `.live`
   - `var showError: Bool` -- returns `true` for `.error`, `.permissionDenied`, `.emptyLibrary`

3. Add an `isFirstLoad` property (`private(set) var isFirstLoad = true`) -- set to `false` after the first successful transition to `.live` or `.cached`.

4. Modify `refreshFromLive(tracks:)`:
   - Accept an `isLoadingTracks: Bool` parameter: `func refreshFromLive(tracks: [Track], isLoadingTracks: Bool)`
   - At the top, add the guard: `guard !isLoadingTracks || !tracks.isEmpty else { return }` -- this is the core bug fix. When `isLoadingTracks=true` and `tracks=[]`, we're still loading so don't transition to `.emptyLibrary`.
   - When transitioning from `.shimmer` to `.live`, enforce minimum shimmer hold:
     ```swift
     // Inside refreshFromLive, after computing metrics, before setting loadingState:
     if loadingState == .shimmer, let startTime = shimmerStartTime {
         let elapsed = Date().timeIntervalSince(startTime)
         let remainingHold = max(0, 0.5 - elapsed)
         if remainingHold > 0 {
             Task {
                 try? await Task.sleep(for: .seconds(remainingHold))
                 loadingState = .live
                 isFirstLoad = false
             }
             return // Don't set loadingState synchronously
         }
     }
     loadingState = .live
     isFirstLoad = false
     ```

5. Add a loading timeout: `private var loadingTimeoutTask: Task<Void, Never>?`. Start a 15-second timer in `loadCachedMetrics` when entering `.shimmer`:
   ```swift
   loadingTimeoutTask?.cancel()
   loadingTimeoutTask = Task {
       try? await Task.sleep(for: .seconds(15))
       if loadingState == .shimmer {
           loadingState = .error("Loading timed out. Please check your Music library access and try again.")
       }
   }
   ```
   Cancel it when transitioning out of shimmer.
