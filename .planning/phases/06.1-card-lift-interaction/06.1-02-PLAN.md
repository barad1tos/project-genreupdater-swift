New section `// MARK: - Card Lift`:

`func liftCard(sourceID: String, contentType: CardContentType, sourceFrame: CGRect)`:
- Creates `CardLiftState(sourceID: sourceID, contentType: contentType, phase: .pressing, sourceFrame: sourceFrame)`.
- Sets `cardLiftState`.
- If `!reduceMotion` (pass from View via parameter or check `NSWorkspace.shared.accessibilityDisplayShouldReduceMotion`): After 150ms delay (`Task.sleep(for: .milliseconds(150))`), transition phase to `.lifted` with spring animation. Use `withAnimation(Motion.cardLiftSpring)` for the phase change.
- If reduceMotion: Set phase to `.lifted` immediately (no animation).

`func dismissCardLift()`:
- If `cardLiftState` has a cascade parent (parentSourceID != nil): restore parent state (create new CardLiftState from parent fields, phase = .lifted).
- Else: set `cardLiftState?.phase = .dismissing`, then after 450ms delay, set `cardLiftState = nil`.
- With reduceMotion: set `cardLiftState = nil` immediately.

`func cascadeToAlbum(album: AlbumSummary, sourceFrame: CGRect)`:
- Save current cardLiftState's sourceID, contentType, sourceFrame as parent fields on a new state.
- Create new CardLiftState with `.album` contentType, the album's sourceFrame, and parent reference fields.
- Transition through pressing -> lifted like liftCard.

`func artistGroupForCardLift(_ name: String) -> ArtistGroup?`:
- Look up the artist in current sections by canonicalName. Return the ArtistGroup.

Add `import SharedUI` to the imports (needed for CardLiftState, CardContentType from SharedUI).

Note: reduceMotion must be passed from the View layer since `@Environment` is not available in ViewModels. Add a `var reduceMotion: Bool = false` property to the ViewModel that the View sets.

**Add DoubleClickDetector overlays to artist rows:**

In `artistRow(_ artist:)`, add `.overlay { DoubleClickDetector { ... } }` on the ArtistListRow. The closure should:
1. NOT trigger if Cmd or Shift is held (check `NSApp.currentEvent?.modifierFlags`).
2. Capture the row's frame using a GeometryReader approach or anchorPreference. For simplicity, use a `@State` dictionary to store frames, or use `GeometryReader` inline to capture the row's frame in the global coordinate space at tap time.
3. Call `viewModel.liftCard(sourceID: artist.id, contentType: .artist(name: artist.canonicalName), sourceFrame: capturedFrame)`.

Recommended approach for frame capture: Add a `PreferenceKey` that collects row frames. Each artist row reports its bounds via `.anchorPreference(key: CardLiftSourceKey.self, value: .bounds)`. At the BrowseView level, read preferences with `.overlayPreferenceValue`. However, since we need the frame at double-click time (not continuously), a simpler approach is to use a `GeometryReader` background on each row that captures the frame into a dictionary `@State private var rowFrames: [String: CGRect] = [:]`. The DoubleClickDetector action reads from this dictionary.

Actually, use the anchorPreference approach from RESEARCH.md as it is cleaner:

1. Define `CardLiftSourceKey: PreferenceKey` (dictionary of String -> Anchor<CGRect>) at the top of BrowseView.swift.
2. On each artist row: `.anchorPreference(key: CardLiftSourceKey.self, value: .bounds) { [artist.id: $0] }`.
3. On each album sub-row: `.anchorPreference(key: CardLiftSourceKey.self, value: .bounds) { [album.id: $0] }`.
4. At the BrowseView body level, wrap everything in a `GeometryReader` and use `.overlayPreferenceValue(CardLiftSourceKey.self)` to resolve anchors to CGRects when the double-click fires.

**Create cardContent helper:**
```swift
@ViewBuilder
private func cardContent(for state: CardLiftState) -> some View {
    switch state.contentType {
    case let .artist(name):
        if let artist = viewModel.artistGroupForCardLift(name) {
            ArtistCardContent(
                artist: artist,
                albums: viewModel.albumsForArtist(name),
                tracks: viewModel.tracksForArtist(name),
                onAlbumDoubleTap: { album in
                    // Cascade: capture frame (use .zero for cascade since card is already centered)
                    viewModel.cascadeToAlbum(album: album, sourceFrame: .zero)
                }
            )
        }
    case let .album(name, artistName):
        let albumID = AlbumIdentifier(albumName: name, artistName: artistName)
        AlbumCardContent(
            album: viewModel.albumSummary(for: albumID) ?? AlbumSummary(name: name, artist: artistName, year: nil, trackCount: 0, primaryGenre: nil, healthRatio: 0),
            tracks: viewModel.tracksForAlbum(albumID)
        )
    }
}
```
